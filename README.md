# FlowTasks
The FlowTask App is a task management web application designed to help users organize both recurring and one-off responsibilities across different areas of life such as school, work, personal, and home. Its main purpose is to reduce mental overload by transforming repetitive commitments into a structured and predictable workflow. Instead of relying on memory, users can simply check the app to stay on track, ensuring important tasks are not forgotten.

FlowTask simplifies weekly planning by allowing users to create tasks with titles, descriptions, categories, and priority levels, while also defining custom frequencies (once, daily, weekly, or custom). Tasks can be scheduled for specific days of the week and are displayed in a calendar-style weekly view with clear visual indicators for pending, completed, and overdue items. Incomplete tasks automatically carry over as overdue until marked as completed, encouraging consistency and accountability.

Technically, FlowTask is built with a Node.js + Express backend following RESTful API principles, a React.js frontend with React Router for navigation, and a PostgreSQL database for persistence. The project is deployed on Render, version-controlled with Git, and developed using VS Code. Together, these technologies provide a modern, reliable, and user-friendly solution for managing daily and weekly responsibilities.
